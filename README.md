## 력습지 및 청년 성장 돕기 - 관리자 페이지
![Logo](https://github.com/tylerkim1/RYUK_Manage/assets/61447161/519e079c-84db-4e3e-b04b-ba9c6516408b)
### 1. 프로젝트 개요
“우리는 모두 언젠가 백수가 된다.”
많은 청년들이 무업 기간을 겪으며 불안, 단절, 우울을 경험합니다. 니트생활자는 무업 기간 사회적 단절을 경험하는 청년들이 연결되는 커뮤니티 플랫폼입니다. 력습지는 이런 무업 청년들의 일상에 찾아오는 무기력 및 우울 관리를 위한 학습지 서비스입니다. 청년들이 무기력의 늪에 빠지지 않도록 경험을 통한 성취감을 고취하고 사회적인 연결을 만드는 활동입니다. 첫 번째로는 할 일을 부여해 소소한 성취감부터 시작해서 다른 것들도 해낼 수 있는 마음상태를 만들고, 두 번째로는 같은 상황에 놓인 청년들 간의 커뮤니티를 형성해 공존을 위한 연결을 만들고, 세 번째로는 시간이 지남에따라 성장하는 과정을 시각화해 무업 기간을 전환 기간으로 바꾸는 것이 목표입니다.

기존에도 수기로 력습지를 진행했으나, 수기로 진행했기 때문에 인력 의존도가 높다는 단점이 있었습니다. 또한, 대화와 미션 완수 모두 카카오톡 채팅방에서 일어나서 사용에 불편함이 있었습니다. 따라서 본 프로젝트는 기존 력습지 서비스를 자동화하여 관리할 수 있는 관리자 웹페이지, 그리고 미션 수행 및 성취를 한눈에 볼 수 있는 사용자 앱으로 나누어서 진행하였습니다.
특히, 이전 력습지 관리자들은 사용자 개인마다 미션 완료 유무를 체크하고, 이에 대해 리마인드를 주기적으로 시행해야 했기에 쓸데 없이 많은 시간을 할애했습니다. 이러한 불편함과 시간 낭비를 줄이기 위해, 웹 형식의 관리자 사이트를 만들었으며, 많은 자료들을 한 눈에 확인 가능하게끔 제작되었습니다. 

------
### 2. 프로젝트 설명
관리자 페이지에서는 팀별로 속해있는 사용자들의 미션 완료 현황을 볼 수 있으며, 새롭게 사용자에게 미션 할당이 가능합니다. 또한, 팀과 사용자의 삭제/추가/수정이 가능한 페이지를 추가로 제작하였습니다. 해당 기능을 구현한 4개의 페이지로 구성되어 있습니다.
#### A. 통계 탭
통계 탭은 사이트의 첫 화면으로 보여지는 탭이며, 사용자들의 미션 달성 현황을 한 눈에 볼 수 있습니다. 
![20231201_010514_4](https://github.com/tylerkim1/RYUK_Manage/assets/125373555/47ca5e0c-88fb-463f-b870-ed5c9c5b8d17)
위의 화면처럼 기간 범위, 사용자 범위, 날짜를 선택하면 해당 선택에 맞는 미션 달성률을 보여줍니다. 미션 달성률이란, 실행해야 하는 전체 미션 개수에 대한 완료된 미션 개수의 퍼센트를 의미합니다. 예를 들어, 사진에서 니트족 No. 98이란 사용자는 총 3개의 미션이 할당되었으며, 3개의 미션을 완료한 상태입니다. 
또한, 데이터 출력 위에 있는 설정 3가지에 따라 보여지는 데이터가 달라집니다.
###### 기간 범위
첫번째 선택 칸은 기간 범위를 설정하는 칸입니다. 이는 드롭다운 형태로 구현되며, 총 2가지 선택지가 존재합니다.
1. 일별 - 하루 치의 통계 자료를 보여줍니다
2. 월별 - 한달 치의 통계 자료를 보여줍니다
###### 사용자 범위
두번째 선택 칸은 사용자 범위를 설정하는 칸입니다. 이 또한, 2개의 선택지를 가진 드롭다운 형태입니다. 
1. 개인별 - 선택 기간에 참여 중인 모든 사용자들의 통계 자료를 각각 보여줍니다
2. 팀별 - 선택 기간에 존재하는 모든 팀별 통계 자료를 각각 보여줍니다. 팀별 통계 자료란, 팀 내에 속해있는 모든 사용자들의 미션 개수를 통합한 데이터를 의미합니다. 아래 화면은 팀별을 선택했을 시 보여지는 화면 예시입니다.

### 3. 실행방법
현재는 서버에 올리지 않은 상태입니다. 그럼으로 다음의 방식을 따라 local 환경에서 실행이 가능합니다.
##### 1. github의 web URL을 "git clone"하여 해당 깃헙을 local에 다운받는다.
##### 2. 터미널에 다음 순서대로 입력한다.

    npm install
    npm start

(사용가능한 데모 링크 (ex. 데모 웹사이트 링크, 실행가능한 Google colab 파일, 데모 앱 설치 링크 …)
